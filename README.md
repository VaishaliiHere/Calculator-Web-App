# Calculator-Web-App
A responsive calculator web app using HTML, CSS, and Javascript with basic operations such as:
## Mathematical operations 
* Addition (+)
* Subtraction (-)
* Multiplication (*)
* Division (/)
* Percentage (%)
* Decimal Calculations
* Square root
* Square

  ## User Control Buttons

* Number Buttons (0–9)
* Decimal Button (.)
* Clear Button (C)
* Backspace/Delete Button (⌫)
* Equal Button (=)
